An autobiography by chatGPT-3

“chatGPT chatGPT chatGPT“

The buzz around the tech world is so noisy that, by now, you probably heard of or played with chatGPT. When I started to learn more about chatGPT and was thinking how chatGPT-3 can help me in my day-to-day life. Well, I am a Software Engineer and hate writing engineering documents. A thought splash in my mind, “Hey, why don’t we ask chatGPT to do the work for me?”

As I stated get my hands dirty on chatGPT-3, as all of us do, I was searching for more info about chatGPT. I decided instead of searching (Googling?), I turned into chatGPT itself. Yes, I decided to ask chatGPT about itself. This is an attempt to make chatGPT write about itself and hence the blog “An autobiography by chatGPT-3”.

First thing first:

I googled chatGPT (Hmm, cannot avoid Google search) and landed on the OpenAPI web site. I navigated to https://beta.openai.com/docs/quickstart and played with the “try now” examples. I signed up a personal account with chatGPT which requires mobile number for verification. Don’t be surprised if you are asked to wait to try your chatGPT “completion”. So many people are trying and sometime chatGPT server must queue the requests. Once you all set with your account, there is a “playground” and I was playing with it.

Time to ask about itself:

First, I asked the chatGPT “Write chatGPT blog”. You might wonder why I used minimal words in the “prompt”. But there is something called “tokens” in NLP beside I am an Engineer too. There is a restriction in using number of tokens. Ie, you pay for number of tokens. The more detailed “prompt” gives accurate answers. The more tokens mean more money.

Now, you can see the content is not enough for a blog! So, I decided not to let chatGPT go and asked more questions from the answer like we do in the IT interview process. Next, I asked “How chatGPT works”. Here is the details of chatGPT and my conversation.

Clearly not enough for a blog. My ‘ego’ did not let chatGPT go and forced the chatGPT to respond for “the potential applications of GPT technologies”.

This back-and-forth conversation between chatGPT and me happened for a while and somewhere I had to stop. The following is the content that chatGPT told me about itself.

“how chatGPT works”

ChatGPT is a natural language processing (NLP) system that can generate human-like conversations. It works by using a deep learning model trained on a large dataset of human conversations, which allows it to understand the context and generate responses that are more relevant to the conversation. The system takes input from the user’s conversation and uses its neural network to generate an appropriate response. The model then selects the most likely response from the list of generated options. Finally, the response is returned to the user for further interaction.

“the various use cases for ChatGPT”

  • Human-Computer Interaction: ChatGPT can be used in a conversational agent or chatbot to provide natural language responses to questions and commands. The bot can also be used to generate personalized conversations with users.
  • Natural Language Processing (NLP) Tasks: ChatGPT can be used for text generation such as summarization, question answering, dialogue generation, story generation, etc.
  • Social Media & Information Retrieval: ChatGPT can be used to generate relevant social media posts, blog entries, and other content related to a given topic. It can also be used to retrieve information from webpages and databases using natural language queries.
  • Productivity Applications: ChatGPT can be used to automate mundane tasks such as writing emails, creating reports, scheduling appointments, etc.
  • Education & Training: ChatGPT can be used to create interactive tutorials, presentations, and simulations that respond to user input in realtime.

“the potential applications of GPT technologies”

GPT technologies can be applied in a variety of areas, including:

  • Natural language processing (NLP): GPT models can be used to improve the accuracy and performance of NLP systems for tasks such as machine translation, text summarization, question answering, and natural language understanding.
  • Autocompletion: GPT models can be used for autocomplete applications that suggest words or phrases as users type.
  • Text generation: GPT models can be used to generate text from scratch, allowing developers to create new stories, articles, poems, and jokes.
